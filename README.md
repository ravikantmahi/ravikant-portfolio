## ✨ Architectural Features & UI Design

This portfolio isn't just a resume; it's a demonstration of modern frontend engineering.

* **Crimson Slate Aesthetic:** A custom-designed color palette featuring deep slate backgrounds, crimson accents, and frosted glass elements (**Glassmorphism**).
* **Bento Box Grid System:** A modern, Apple-inspired asymmetrical grid layout for cleanly categorizing the core tech stack.
* **Fluid Responsiveness:** Utilizes modern CSS mathematical functions (`clamp()`, `min()`, `minmax()`, `aspect-ratio`) to ensure a flawless, pixel-perfect scaling experience on screens ranging from 4K ultra-wide monitors to the narrowest mobile devices.
* **Cinematic Animations:** Custom CSS `@keyframes`, smooth Vanilla JS `IntersectionObserver` scroll reveals, interactive hover states, and dynamic typing effects.
* **Serverless Contact Architecture:** Integrated with `FormSubmit` to securely route messages directly to my inbox without requiring a dedicated backend server.
